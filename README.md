# Sobre o que é esse repositório

Este repositório foi criado apenas para reprodução dos testes de automatização com cypress 
proposto no Curso "Testes automatizados com Cypress - Básico" do Professor Walmyr Filho, 
adquirido na Udemy. 

Os testes foram escritos no arquivo CAC-TAT.spec.js, e toda a metodologia do curso, juntamente
com as indicações do professor estão nos arquivos .md dentro da pasta lessons e no arquivo 
"Informações Sobre o Projeto Proposto".
